# Router
/ -> Home
/join -> Sign up
/login -> Login
/search -> Search

/user/:id -> See User
/user/logout -> Log out
/user/:id/delete -> Delete User
/user/:id/edit -> Edit My Profile


/video/:id -> See Video
/video/:id/edit -> Edit Video
/video/:id/delete -> Delete Video
/video/upload -> Upload Video
